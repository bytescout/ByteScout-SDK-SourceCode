##### **PDFFromHTML.cmd:**
    
```
curl --location --request POST 'https://api.pdf.co/v1/pdf/convert/from/html' \
--header 'x-api-key: {{x-api-key}}' \
--form 'html=# Hello World!' \
--form 'name=result.pdf'
```
